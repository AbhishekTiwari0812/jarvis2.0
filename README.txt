JARVIS 2.0
Audio to text.
Get current news.
Get jokes,emails,poetry.
Filter emails (semi-automatic).
------------------------------------------
tested on python 2.7
Installations needeed:
Speech recognition lib:
python -m pip install SpeechRecognition
------------------------------------------
Need to provide your own API KEY from newsapi.org. 
Add it in the API_KEY.txt file inside data folder:
news_org_api: xxxxxxxxxxxxxxxxxx
